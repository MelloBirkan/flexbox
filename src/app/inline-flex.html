Flexbox

Display: Flex

Qualquer elemento pode se tornar um container flexível. Containers flexíveis são ferramentas úteis para criar sites que se adaptam a diferentes tamanhos de tela. Os elementos filhos de um container flexível ajustam seu tamanho e posição conforme o tamanho e a posição do container pai. Para que um elemento se torne um container flexível, a propriedade display deve ser configurada como flex:

div.container {
display: flex;
}

No exemplo acima, todas as divs com a classe container são containers flexíveis. Se elas tiverem filhos, esses filhos serão itens flexíveis. Uma div com a declaração display: flex; permanecerá como um elemento de nível de bloco — ou seja, nenhum outro elemento aparecerá na mesma linha que ela. No entanto, isso alterará o comportamento de seus elementos filhos. Os elementos filhos não iniciarão em novas linhas. Nos exercícios a seguir, veremos como a propriedade display: flex; afeta o posicionamento dos elementos filhos.
